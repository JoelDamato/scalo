Objetivo

Pasar de un registro operativo
a una herramienta de gestión.

La mejora busca que Studio Bloom no mire solo ventas. También necesita contemplar comisiones variables, gastos diarios, insumos, comodidades para clientas y costos fijos para saber si el negocio realmente está siendo rentable.

Sistema actual

Mucho movimiento,
poca visibilidad real

  • Las ventas se registran, pero no muestran la rentabilidad real.
  • Las comisiones se calculan de forma general y obligan a revisar manualmente.
  • Los gastos chicos del día a día se pierden y terminan afectando el margen.
  • No hay una vista simple que conecte ingresos, costos y ganancia.
Sistema propuesto

Cada servicio, cada gasto
y cada peso bajo control

  • Comisiones variables por servicio y profesional, sin cálculos a mano.
  • Registro ordenado de insumos, comodidades y gastos operativos diarios.
  • Vista clara de gastos fijos para saber cuánto hay que facturar antes de ganar.
  • Dashboard financiero con rentabilidad mensual y métricas accionables.
Propuesta de mejora

Cinco cambios concretos
para ordenar el negocio.

La idea no es sumar complejidad. Es hacer visible lo que hoy impacta en el resultado del studio pero todavía no está bien medido dentro del sistema.

1. Comisiones variables

Cada servicio podría tener una lógica distinta según el trabajo real de cada profesional.

Servicio realizado Profesional asignada Fijo o porcentaje Excepciones
  • ✓Configurar comisión específica por combinación de servicio y empleada.
  • ✓Usar monto fijo o porcentaje según corresponda.
  • ✓Evitar discusiones y liquidaciones manuales a fin de mes.

2. Insumos y comodidades

Crear una sección nueva para cargar todo lo que se usa para brindar el servicio y que hoy suele quedar fuera del análisis.

Papel higiénico Toallitas Agua, café y té Cremas y descartables
  • ✓Nombre del insumo, categoría, fecha y monto.
  • ✓Frecuencia: diaria, semanal, mensual o eventual.
  • ✓Observación opcional para contexto operativo.

3. Gastos fijos del negocio

Una vista mensual para tener presentes los costos que existen aunque no entre una sola venta.

Alquiler Servicios Sueldos Publicidad
  • ✓Control de gastos recurrentes y extraordinarios.
  • ✓Lectura rápida del punto de equilibrio mensual.
  • ✓Más claridad antes de hablar de ganancia.

4. Rentabilidad real

Una vista financiera simple para entender qué está dejando dinero y qué no.

  • ✓Ventas totales del mes.
  • ✓Total en comisiones, gastos fijos e insumos.
  • ✓Ganancia estimada y margen de rentabilidad.
  • ✓Servicio, profesional y box más rentables.
